# req-url-extra by Richard Cook

Haskell application with main executable only built using Stack

## Clone repository

```
git clone https://github.com/rcook/req-url-extra.git
```

## Install compiler

```
stack setup
```

## Build

```
stack build --fast
```

## Run application

```
stack exec req-url-extra-app
```

## Run tests

```
stack test
```
